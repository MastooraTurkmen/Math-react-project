# Math-o-Matic

Warning: Vite enforces using jsx syntax inside jsx/tsx files, so it will complain about that. Solution: rename `.js` files to `.jsx` :)

Quick start:

```
$ npm install
$ npm start
````

```
$ pnpm install
$ pnpm start
````


Head over to https://vitejs.dev/ to learn more about using vite
